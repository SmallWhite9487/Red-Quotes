# 🇨🇳 红语录 (Red Quotes)

> **星星之火，可以燎原。**

## 📖 项目简介

**红语录** 是一款致力于弘扬红色文化、传承革命精神的桌面小程序。本项目灵感源自《毛主席语录》，旨在汇集中华人民共和国历代领袖的经典名言与思想精华。

通过数字化形式，我们希望让更多人能够便捷地学习与重温那些激荡人心的历史强音，在**毛泽东思想**、**邓小平理论**、**“三个代表”重要思想**、**科学发展观**以及**习近平新时代中国特色社会主义思想**的指引下，时刻警醒自我、激励自我，培养自强不息的品格，争做社会主义事业的合格建设者与接班人！

## 🌟 主要功能

### 🚩 <每日一语>
不仅是语录的展示，更是精神的洗礼。支持按领导人分类查看或随机抽取经典语录。

收录领袖包括：
*   **毛主席**  
*   **刘主席**  
*   **邓公**  
*   **江主席**  
*   **胡主席**    
*   **习主席**  

### 🎵 <红色电台> (**WIP**)
*Work In Progress*
计划收录经典红色歌曲，让激昂的旋律伴随左右，陶冶情操，振奋精神。
